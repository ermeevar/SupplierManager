# SupplierManagerTutor

Приложение для просмотра продукции компании "Лопушок".

## Installation

Скачать инсталлятор SupplierManagerInstallator. Распаковать его.



## Описание

Компания производит туалетную бумагу под брендом “Лопушок”. На протяжении
нескольких лет выпускает бумагу различной плотности и с разным количеством слоев. Система предоставляет возможномсть храненить информацию о продукции.
